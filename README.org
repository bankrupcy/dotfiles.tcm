** emacs

=Installation=

#+begin_src
brew install emacs-plus --HEAD --with-modern-doom3-icon --with-no-titlebar
#+end_src

Native comp feels slower on M1 in my opinion.

=Post Installation=

#+begin_src
stow ~/.dotfiles/emacs
launchctl load -w ~/Library/LaunchAgents/gnu.emacs.daemon.plist
#+end_src
